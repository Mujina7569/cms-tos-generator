# CMS ToS Generator

Terms of Service Generator for Commission.

[For English](README_EN.md)

## Description

ด้วยความที่เจอกับคำถามถึงการเริ่มคอมมิชชั่นมากมาย ตั้งแต่การเตรียมตัว ช่องทางการชำระเงิน การเขียนข้อตกลงต่างๆ อยู่บ่อยครั้ง ผมจึงคิดที่จะสร้างเครื่องมือช่วยอำนวยความสะดวกนี้ขึ้นมา

เพียงแค่กรอกข้อมูล ข้อตกลงก็จะพร้อมใช้งาน!
